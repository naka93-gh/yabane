# Yabane

[![CI](https://github.com/naka93-gh/yabane/actions/workflows/ci.yml/badge.svg)](https://github.com/naka93-gh/yabane/actions/workflows/ci.yml)

プロジェクト計画管理デスクトップアプリ。

## セットアップ

```bash
npm install
npm run dev
```

## コマンド

| コマンド               | 内容                      |
| ---------------------- | ------------------------- |
| `npm run dev`          | 開発サーバー起動          |
| `npm run build`        | 型チェック + ビルド       |
| `npm run build:win`    | Windows 向けパッケージング |
| `npm run build:mac`    | macOS 向けパッケージング   |
| `npm run build:linux`  | Linux 向けパッケージング   |
| `npm run lint`         | ESLint                    |
| `npm run format`       | Prettier                  |
| `npm run typecheck`    | 型チェックのみ            |
